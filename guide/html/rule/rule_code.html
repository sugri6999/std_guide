코드규칙

Publishing Conventions

기본규칙

코드 들여쓰기

  • 1탭 들여쓰기 하며, 1탭의 크기는 공백 4칸으로 설정한다.

들여쓰기를 하지 않는 경우

								<!doctype html>
								<html lang="ko">
								<head>
								</head>
								<body>
								<div id="wrapper">
								</div>
								</body>
								</html>

주석 / 빈줄

  • 가독성을 고려하여 빈 줄을 추가한다. 단, 1줄을 초과하지 않는다.
								<header id="header">
								</header> <!-- //Header -->

								<div id="container">
								</div> <!-- //Container -->

								<footer id="footer">
								</footer> <!-- //Footer -->

수정/추가/삭제

  • 조치사항 진행 결과의 주석을 남긴다. (예 : [날짜][조치내용][이슈])
  • 잦은 수정사항을 고려하여 다중구조인 경우만 닫기주석으로 범위를 표시한다.

인라인 수정

								<div>
									(내용생략) <!-- 181130 금리변경건 수정 -->
								</div>

단순구조 수정

								<div>
									(내용생략)
								</div> <!-- //181130 금리변경건 수정 -->

다중구조 수정

								<!-- 181130 금리변경건 수정 -->
								<div>
									(내용생략)
								</div>
								<div>
									(내용생략)
								</div>
								<!-- //181130 금리변경건 수정 -->

이슈단위 표시

  • 조치사항 이슈가 많은경우 이슈단위로 주석을 남긴다. (예 : [제목][조치내용][이슈])
								<!-- 접근성ver1 제목문구 수정 -->
								<div title="제목">
									(내용생략)
								</div>

Attribute 지정 순서

기능 > 이름 > 정보 > 선택속성 > 클래스 > 상태 순서로 적용한다.

A 엘리먼트

<a href="#none" title="새창열기" target="" class="">인쇄하기</a>

Iframe 엘리먼트

<iframe src="about:blank" title="빈프레임"></iframe>

Img 엘리먼트

<img src="img.gif" alt="대체텍스트" />

Input 엘리먼트

<input type="text" name="" id="" value="" title="" class="" style="" disabled />

CSS 기본 규칙

작업정보

							@charset "utf-8";
							/*-------------------------------------------------------------------
								파일정의 : 파일명 또는 파일설명
								작성날짜 : 2019-01-01 홍길동
								주의사항 : 주석표기는 페이지 접근방법을 알수 있도록 표시
								분류순서 : 분류1 > 분류2 > 분류3
								속성순서 : 표시 > 위치 > 넘침/흐름 > 크기/간격 > 박스모양 > 폰트/정렬 > 기타
							-------------------------------------------------------------------*/
  • * 파일에 따라서 불필요한 항목은 삭제 할 수 있다. (예 : animate.css)

작은 따옴표('') 사용 범위

폰트의 선언, filter, content에 작은 따옴표('')를 사용한다.

								.class-name {font-family:'dotum'}
								.filter:progid:DXImageTransform.Microsoft.AlphaImageLoader(src='img.png', sizingMethod='scale'}
								.class-name:after{content:'.'}
								.class-name {background:url('img.gif') no-repeat}

세미콜론(;)

마지막 선언된 속성에는 세미콜론(;)을 사용하지 아니한다.

								.class-name {margin-left:0; padding-left:10px}

공백의 사용

선택자 간, 중괄호간 한칸의 공백을 사용한다, 중괄호 안쪽 좌우에는 공백은 사용하지 않는다.

								.class-name,
								.class-name .child-class {background:url('../img/img.gif') no-repeat}
								.class-name {padding:0; margin:0}
								.class-name {padding:0; margin:0}

들여쓰기/줄 바꿈

  • CSS 코드 작성시 들여쓰기는 허용하지 않는다.
  • 공통으로 사용되는 속성의 선택자간의 줄 바꿈은 허용하며 이 외의 줄바꿈은 허용하지 않는다.
  • 하나의 속성이 끝나고 다음속성은 한칸 띄우고 선언한다.
								h4.class-name {font-size:1.25em; letter-spacing:-1px; margin:0.5em 0 0 0;}

								.class-name,
								.class-name .child-class {background:url(../img/img.gif) no-repeat;}

주석 / 빈줄

  • 종료 주석은 작성하지 않는다.
  • 코드 그룹 간 1줄의 빈 줄을 허용한다. 단, 1줄을 초과하지 않는다.
								/* title */
								.title-type1 {margin:30px 0 10px 0}

								/* description */
								.desc1 {padding:2px 0 0 15px; font-size:0.917em}

선택자

공통 선택자

'*' 공통 선택자는 웹 페이지의 성능을 저하를 가져오기 때문에 가급적 사용하지 않는다.

								/* 잘못된 예 */
								* {margin:0; padding:0}
